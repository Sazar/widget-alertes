# 🎮 Widget Alertes Twitch — StreamElements

Widget d'alertes moderne et épuré pour StreamElements. Affiche les **Subs**, **Resubs**, **Gifted Subs**, **Followers**, **Dons (Tips)** et **Bits** avec animations fluides et hautement configurable.

## 📁 Fichiers

| Fichier | Rôle |
|---|---|
| `widget.html` | Structure HTML du widget |
| `widget.css` | Styles & animations |
| `widget.js` | Logique des alertes & événements SE |
| `widget.fields.json` | Paramètres configurables dans SE |

## 🚀 Installation

1. Dans StreamElements, créer un widget **Custom Widget**
2. Copier le contenu de chaque fichier dans l'onglet correspondant
3. Importer `widget.fields.json` dans l'onglet **Fields**
4. Personnaliser les paramètres selon vos goûts

## ✨ Fonctionnalités

- Alertes animées : Sub, Resub, Gift Sub, Follow, Tip, Bits
- Queue d'alertes (pas de chevauchement)
- Animations d'entrée/sortie configurables
- Couleurs, polices, tailles entièrement personnalisables
- Sons activables/désactivables par type d'alerte
- Effets visuels : particules, shake, flash, confetti
- Aperçu test intégré
